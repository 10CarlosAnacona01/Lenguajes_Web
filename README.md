# Lenguajes_Web

Proyecto de una aplicación web para un Spa de Uñas desarrollado con React y Vite.

## Descripción

Esta aplicación busca ofrecer una interfaz moderna para visualizar y gestionar servicios de un spa de uñas.

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- UIkit
- @emotion/styled

## Instalación y ejecución

1. Clonar el repositorio
2. Instalar dependencias:

npm install

3. Ejecutar el proyecto:

npm run dev
